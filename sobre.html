Sobre o MyFinance

Conheça o projeto

📌 O Problema

Muitas pessoas não conseguem controlar suas finanças pessoais. Gastam mais do que ganham e não sabem para onde o dinheiro está indo. A falta de organização financeira é um problema real que afeta milhões de brasileiros.

💡 A Solução

O MyFinance é um sistema simples e intuitivo para controle de finanças pessoais. Ele permite registrar receitas e despesas, visualizar o saldo, filtrar transações e ter uma visão clara da saúde financeira.

🛠️ Tecnologias

HTML5 CSS3 TypeScript Vite LocalStorage Git GitHub Pages

📚 Conceitos Utilizados

  • Programação Orientada a Objetos (Classes)
  • Funções e Modularização
  • Arrays e Métodos de Array (map, filter, reduce)
  • Estruturas Condicionais (if, else, switch)
  • Estruturas de Repetição (for, for...of)
  • Manipulação do DOM e Eventos
  • LocalStorage
  • Responsividade (Media Queries)
  • TypeScript (Tipagem)

👤 Desenvolvedor

Projeto desenvolvido como trabalho acadêmico para as disciplinas de Programação II e Construção de Sites. O sistema utiliza arquitetura de Múltiplas Páginas (MPA) para aplicar conceitos sólidos de interligação HTML.
